# Loan Approval Prediction

## Project Overview

The goal of this project is to predict whether a loan application will be approved or rejected using machine learning models.

The notebook includes:

* Exploratory Data Analysis (EDA)
* Data preprocessing
* Feature engineering
* Model training
* Hyperparameter tuning using RandomizedSearchCV
* Model evaluation using:

  * Accuracy
  * Precision
  * Recall
  * F1-Score
  * ROC-AUC
  * Confusion Matrix
* Comparison between multiple machine learning algorithms

## Models Used

* Logistic Regression
* Random Forest
* XGBoost

* Python
* Pandas

* Plotly
